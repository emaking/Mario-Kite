## Controls :
* Kite :
  * Numpad 7 :   left turn
  * Numpad 9 :   right turn

* Board
  * Numpad 4 : left turn
  * Numpad 1 : strong left turn
  * Numpad 6 : right turn
  * Numpad 3 : strong right turn

* Mario
  * Numpad 5 : Jump
  * Numpad + : Fly

* Game 
  * F1 : play / pause
  * Escape : Quit

* Camera
  * 1 : Look right
  * 2 : Look left
  * 3 : Look up
  * 4 : Look down
  * F3 : Move down
  * F4 : Move up
  * F5 : Move back wind
  * F6 : Move front wind
  * F7 : Move right
  * F8 : Move left
  * F12 : capture bmp image
