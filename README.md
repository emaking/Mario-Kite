## Install
  
- Download & unzip archive
- Double click MarioKite.exe
  
  
## Usage 
  
* **Game** 
  * F1 : Play / Pause
  * Escape : Quit
  
  
* **Kite**
  * Numpad 7 : Left turn
  * Numpad 9 : Right turn
  
  
* **Board**
  * Numpad 4 : Left turn
  * Numpad 1 : Strong left turn
  * Numpad 6 : Right turn
  * Numpad 3 : Strong right turn
  
  
* **Mario**
  * Numpad 5 : Jump
  * Numpad + : Fly 
  
  
* **Camera**
  * 1 : Look right
  * 2 : Look left
  * 3 : Look up
  * 4 : Look down
  * F3 : Move down
  * F4 : Move up
  * F5 : Move back wind
  * F6 : Move front wind
  * F7 : Move right
  * F8 : Move left
  * F12 : Capture image to bmp (_named "0" in root folder_)
